## Como rodar este código

```bash
#clone este repositório

$git clone https://github.com/cmilesnts/cartao-de-perfil.git

#acesse a pasta do projeto no seu terminal 

$cd cartao de perfil 

#intale as dependências

$npm install 

#execute a aplicação

$npm run start
```
